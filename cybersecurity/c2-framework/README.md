# What is it?
Kakerou is an open source C2 framework for security testing. It includes a console, GUI, server, and agents to simulate attacks. The main purpose is to educate myself on how adversaries maintain persistence, avoid detection, and establish a connection. The communication will cover HTTP/S, DNS, SMB, etc. The framework utilizes cryptography to encrypt communication. 

The server and client are configurable through a yaml/json (haven't decided) file that dictates keys, communication method, agent names, etc. These settings are supported in Windows and Linux.
